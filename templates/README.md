# Reference-Card Templates

Use these templates to keep new knowledge-base articles consistent:

- [`function_method_template.md`](function_method_template.md) for functions and methods
- [`data_types_template.md`](data_types_template.md) for data types
- [`module_template.md`](module_template.md) for modules and libraries

Copy the closest template into the correct `docs/` topic folder, then replace
the example metadata and prompts. Preserve the YAML front-matter structure so
future tooling can index every card consistently.
