# Mathematics

Mathematical foundations and engineering-focused worked examples.

- `arithmetic/` - numbers, operations, ratios, percentages, and units
- `algebra/` - equations, expressions, functions, and rearranging formulas
- `geometry/` - shapes, dimensions, area, volume, and coordinate geometry
- `trigonometry/` - angles, triangles, waves, and engineering applications
- `visualizations/` - plots and visual explanations of mathematical ideas
